Countdown - CLI Tool

The countdown command runs a timer for the specified duration. Durations can be specified in seconds (10s), minutes (5m), or hours (1h). You can combine the duration with optional flags to customize behavior.

Installation

Using Homebrew:
brew tap athulquintype/tap
brew install athulquintype/countdown
