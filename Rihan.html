English-calc - CLI Tool

simple English calculator for arithmetic expressions command evaluates short English arithmetic phrases like "three plus five" or numeric expressions like "3 plus 5" and prints the result.Supported operators: plus, minus, times, divided

Installation

Using Homebrew:
brew tap rihan227/english-calc
brew install rihan227/english-calc/english-calc
